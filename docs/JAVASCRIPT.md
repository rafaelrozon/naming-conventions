# JavaScript Naming Conventions

## Filenames

| What            | Rule                              | Example                 |
| --------------- | --------------------------------- | ----------------------- |
| React Component | `[ComponentName].(jsx\|tsx)`      | Avatar.jsx, NavMenu.tsx |
| Selectors       | `[modulename].selectors.(js\|ts)` | cart.selectors.js       |
| Redux Reducer   | `[moduleName].reducer.(js\|ts)`   | cart.reducer.js         |
